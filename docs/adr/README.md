# Architecture Decision Records (ADRs)

We use ADRs to capture significant technical decisions and their context.

## Naming
- Format: `NNNN-title.md` (e.g., `0001-storage-and-cli-first.md`).

## Template
```
# Title

## Status
Proposed | Accepted | Rejected | Superseded

## Context

## Decision

## Consequences
```
